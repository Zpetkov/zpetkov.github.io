ZPETKOV

JSON object generator from text for Java

Information:

The last token of each line is taken as a JSON property name.
If last token is a parseable number the previous token is selected and generated as the number.
                someNumber 123            { "someNumber": 123 }
                
Copy pasting field information from Java classes provides a hint about property type.

            private final String name;
            private final Long id;
            private final Date registerDate;

            Should produce:

            {
                "name": "string",
                "id": 100000,
                "registerDate": "2019-05-05T18:04:04.334Z"
            }
                
Java annotations are ignored
